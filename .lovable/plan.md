

# Update Browser Favicon

## Change
Copy the uploaded logo image to `public/favicon.png` and update `index.html` to reference it as the favicon.

## Steps
1. Copy `user-uploads://image-5.png` → `public/favicon.png`
2. Add `<link rel="icon" href="/favicon.png" type="image/png">` to the `<head>` in `index.html`
